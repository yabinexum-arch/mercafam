🧪 Simulador Wompi Solo pruebas
Simula un pago aprobado de Wompi para probar el webhook completo
¿Qué hace esta prueba?
Envía al webhook exactamente el mismo JSON que Wompi envía cuando un pago es aprobado. Si todo está bien:
1. El webhook genera el código de licencia
2. Lo guarda en Supabase
3. Envía el código por WhatsApp al número ingresado
